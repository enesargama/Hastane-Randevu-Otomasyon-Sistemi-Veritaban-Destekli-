# 🏥 Hastane Randevu Otomasyon Sistemi (PRO)

## 📌 Proje Hakkında
Bu proje, hastaların randevu oluşturabildiği, düzenleyebildiği ve silebildiği,
login sistemine sahip veritabanı destekli bir hastane otomasyon sistemidir.

## 🚀 Özellikler
- Kullanıcı giriş sistemi (login)
- Randevu ekleme
- Listeleme
- Güncelleme
- Silme (CRUD)
- Modern web arayüz

## 🧱 Teknolojiler
PHP, MySQL, HTML, CSS, JavaScript
